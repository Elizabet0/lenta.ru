# Классификация новостей lenta.ru

## Описание проекта
Проект направлен на определение топика новостной статьи на основе заголовка и текста. Использован датасет Lenta.ru с новостями за 1999-2019 годы. Для экспериментов выбраны 5 наиболее частых типа: Россия, Мир, Экономика, Спорт, Культура и 12000 статей по ним. Были реализованы 3 подхода: Tf-Idf + LogReg, Bert и FastText.
Ccылка на датасет: https://www.kaggle.com/datasets/yutkin/corpus-of-russian-news-articles-from-lenta

## Реализованные подходы

1. TF-IDF + Logistic Regression

--Предобработка датасета (токен., стоп-слова, лемматизация с pymystem3)

--Векторизация текста с помощью TfidfVectorizer и последующий лог рег.

--Подбор гиперпараметров для TfidfVectorizer и Logistic Regression выполнен с помощью Optuna (оценка происходила на основе TimeSeriesSplit)

Результат: 85.3% accuracy 

2. RuBERT (+Dropout + Linear)

-- Использована предобученная модель 'DeepPavlov/rubert-base-cased'.

-- К тексту применена токенизация BERT (max_len=256).

-- Добавила в конце dropout + nn.Linear на 5 классов.

-- Обучение: оптимизатор AdamW (lr=3e-5), 6 эпох, сохранение лучшей модели по валидационной выборке

Результат: 91.5% accuracy 

3. FastText + усреднение векторов 

-- Использована cc.ru.300.bin 

-- Каждая статья преобразуется в средний вектор входящих в неё слов

-- Классификатор - логистическая регрессия.

Результат: 81.2 % accuracy
